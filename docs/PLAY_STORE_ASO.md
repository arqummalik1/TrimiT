## 1. How Play Store “SEO” actually works

Google Play has **no separate keyword field** (unlike Apple’s 100-char keyword box). Ranking signals include:

| Signal | What you control |
|--------|------------------|
| **App title** (~30 chars) | Brand + strongest keyword |
| **Short description** (80 chars) | 2–3 primary keywords, readable |
| **Full description** (4,000 chars) | Natural repeats of theme clusters |
| **Category + tags** | Beauty + relevant tags |
| **Installs, retention, ratings** | Product quality + reviews |
| **Crash rate / policy** | Stable builds |
| **Localized listings** | hi-IN + en-IN often beat English-only in India |
| **Updates** | Regular releases signal active app |

**Reality check:** A new app will **not** outrank Fresha, Booksy, or decade-old salon apps on day one. ASO gets you **indexed and competitive** on long-tail queries (“salon booking app india”, “book parlour appointment”) within weeks; head terms take **months + reviews + marketing**.

## 2. Keyword strategy

### 2.1 Keyword clusters (use across title, short, full, screenshots)

| Priority | Cluster | Example search phrases |
|----------|---------|-------------------------|
| **P0** | Salon booking | salon booking app, book salon appointment, salon appointment app |
| **P0** | Salon app (generic) | salon app, salon apps, beauty salon app |
| **P1** | Salon management | salon management app, salon owner app, appointment management salon |
| **P1** | Service type | hair salon booking, barber booking app, beauty parlour booking, spa appointment |
| **P2** | Intent / local | book haircut near me, parlour booking app, salon near me app |
| **Brand** | TrimiT | trimit, trimi t salon |

### 2.2 Do / don’t

| Do | Don’t |
|----|--------|
| Put **Salon Booking** in title or first line of short description | Repeat “salon salon salon” — spam risk |
| Use **owner** + **customer** in full description (dual-sided app) | Claim “#1” or “best” without proof |
| Add **Hindi listing** for India | Stuff competitor names in description |
| Match screenshot text to keywords | Mislead (e.g. “free online payment” if v1 is pay-at-salon) |
| Ask happy users for **5★ reviews** after completed booking | Buy fake reviews (ban risk) |

### 2.3 Competitor terms (strategy only — do not put in listing)

Monitor: Fresha, Booksy, Vagaro, Salonist, MioSalon, Zenoti (India). Note their titles, screenshot hooks, and review keywords. Differentiate: **India-first**, **owner + customer one app**, **pay at salon** for v1.

## 5. Play Console settings beyond text

### 5.1 Category

| Setting | Recommendation |
|---------|----------------|
| **Category** | **Beauty** (primary discovery for salon/beauty searches) |
| **Tags** | Salon, Beauty, Lifestyle, Business, Personalization (max 5) |

### 5.2 Graphic ASO (high conversion → better rank over time)

| Asset | ASO tip |
|-------|---------|
| **Icon** | Clear “T” / brand; readable at 48dp; no tiny text |
| **Feature graphic** | “Salon Booking App” + phone mockup |
| **Screenshots 1–2** | Customer booking flow + keyword headline |
| **Screenshots 3–4** | Map / discover + “nearby salons” |
| **Screenshots 5–6** | Owner dashboard — captures “salon management” searches |
| **Video** (optional) | 30s: search → book → owner notification |

### 5.3 Developer trust

- Fill **developer email**, **website**, **privacy policy**
- Respond to reviews within 48h (keyword-rich replies help sentiment, not index directly)

Example review reply:

> Thanks for using TrimiT for your salon booking! We're glad booking your appointment was easy. If anything failed with slots or notifications, email admin@audentix.com — we'll fix it in the next update.

## 6. Post-launch ASO (weeks 1–12)

| Week | Action |
|------|--------|
| 1 | Ship with listing from LISTING_COPY; enable hi-IN |
| 2–4 | Ask 10–20 real customers + 5 owners for honest 5★ reviews |
| 4 | Check Play Console → **Grow → Store performance** (if available) for search terms |
| 6 | A/B test **short description** only (one change at a time) |
| 8 | Add 2 screenshots if a feature shipped (reschedule, reviews) |
| 12 | Consider **custom store listing** for one city campaign |

**External SEO (helps branded search, not Play algorithm much):**

- Link `https://trimi-t.vercel.app` from Instagram/Facebook salon posts
- Google Business Profile for your **company**, not replacing salon listings
- Press / local beauty bloggers with “TrimiT salon booking app”

## 7. Metrics to track

| Metric | Tool |
|--------|------|
| Store listing visitors → installs | Play Console → Acquisition |
| Search vs browse installs | Play Console → User acquisition |
| Rating average & volume | Play Console → Ratings |
| Uninstall rate | Android vitals |
| Keyword rank (estimated) | AppTweak, Sensor Tower, or manual incognito searches |

**Target (realistic 90 days):** 4.0+ stars, 50+ reviews, indexed for “trimit salon”, long-tail “salon booking app” in your launch cities.

## 8. Compliance (ASO + policy)

- Description must match **actual v1 features** (cash at salon if online pay disabled).
- No guaranteed ranking promises in ads.
- **18+** target audience aligns with booking/account app.
- Data Safety form must match privacy policy (location, photos for owners).
